HTML Tags

About HTML

  • HTML Stands For Hyper Text MarkUp Language
  • HTML is a standard markup language for documents designed in a web Browser
  • HTML Element are the building Blocks of HTML Pages
  • HTML provides a means to create structured documents by denoting structural semantics
  • HTML elements are delineated by tags , written using angle brackets

Tags Used in HTML

  1. <h1> The h1 Tag is Used to Create heading in HTML and their are 6 heading tags from h1 to h6
  2. <p> The p Tag is Used to Display a Paragraph in a Web Page .
  3. <br> The br Tag is Used to break line or to shift to next Line .
  4. <hr> The hr Tag is Used to Create a Horizontal break by Creating a Horizontal Line on the WebPage
  5. <table> The Table Tag is Used to tables in web Pages
